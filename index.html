Discover Our Innovative Telegram Bot Solutions

DanTeng Group specializes in creating powerful tools and solutions based on the popular messaging platform, Telegram Bot. Our products include Notice, a versatile HTTP API for sending text messages to specific Telegram channels, and Word-Map, an intuitive website that collects and organizes users' new words based on their familiarity levels.

Our Products

Notice

Send text messages to a specified Telegram channel through a simple HTTP API. Seamless integration with various web applications, enhanced by CORS support.

Word-Map

An interactive website that stores and categorizes newly encountered words based on familiarity levels, making it an invaluable resource for language learners.
